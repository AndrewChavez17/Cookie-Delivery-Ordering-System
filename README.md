# Cookie Delivery Ordering System

A command-line Python application that simulates a cookie delivery ordering service. Users can select items from a menu, build an order, and receive a total cost with an optional rewards discount.

## Features
- Interactive text-based menu
- Allows users to add multiple items to an order
- Calculates total cost dynamically
- Applies a 10% discount for rewards members
- Displays a summary of items ordered

## Technologies Used
- Python
- Lists
- Loops
- Conditional statements
- User input handling
